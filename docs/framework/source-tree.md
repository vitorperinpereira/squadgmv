## Intent

- `apps/runtime`: API and worker entrypoints
- `packages/*`: shared runtime modules and adapters
- `docs/framework`: always-loaded project guidance
- `docs/stories`: local implementation stories mirrored from planning
- `tests`: unit and integration coverage for the runtime foundation
