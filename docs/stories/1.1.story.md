# Story 1.1: Bootstrap Runtime Shell and Foundational Contracts

**ID:** 1.1 | **Epic:** EPIC-1
**Sprint:** 1 | **Points:** 8 | **Priority:** High
**Created:** 2026-03-08
**Status:** Done

---

## User Story

**Como** Vitor Perin,  
**Quero** uma base executavel do runtime agent-first,  
**Para que** a GMV consiga iniciar missoes, sincronizar planejamento e preparar a execucao dos squads.

---

## Acceptance Criteria

- [x] AC1: o repositorio possui bootstrap de monorepo com `pnpm workspaces + turbo`
- [x] AC2: existe um runtime com modos `api` e `worker`
- [x] AC3: as entidades centrais e contratos basicos estao implementados em pacotes compartilhados
- [x] AC4: o runtime expoe `POST /api/missions`, `GET /api/missions/:missionId`, `GET /health`, boards e endpoints basicos de sync, handoff e approval
- [x] AC5: existe espelhamento local de stories prontas a partir da camada de planejamento
- [x] AC6: lint, typecheck e testes automatizados podem ser executados a partir da raiz

---

## Scope

Inclui a fundacao tecnica das Fases 1 a 3 em nivel bootstrap, com adapters locais para desenvolvimento e interfaces prontas para PostgreSQL, Notion e `pg-boss`.
