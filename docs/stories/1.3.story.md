## User Story

**Como** OpenClaw,  
**Quero** criar projetos no Notion a partir das missoes recebidas,  
**Para que** estrategia vire trabalho executavel com ownership claro.

---

## Acceptance Criteria

- [x] AC1: existe projection de missao para projeto com campos obrigatorios
- [x] AC2: o adapter conhece as quatro databases canonicas do Notion
- [x] AC3: o runtime salva o vinculo de projeto Notion na missao quando a projection ocorre
- [x] AC4: o fluxo foi validado com credenciais reais e databases reais do Notion

---

## Dev Notes

- `PlanningService.projectMissionToPlanningSystem` faz a projection.
- O adapter oficial usa `Projects`, `Epics`, `Stories` e `Tasks`.
- Sem `NOTION_TOKEN` e IDs das databases, o runtime cai no adapter desabilitado com aviso explicito.
- O contrato canonico live de integracao Notion/runtime esta em `docs/notion-live-contract.md`.
